合同保全（债权人的防御性权利体系）
│
├── 代位权（535-537条）
│     │
│     ├── 针对：债务人【消极不作为】——怠于行使权利
│     │
│     ├── 行使条件（四要件）
│     │     ① 债权人对债务人有到期债权
│     │     ② 债务人怠于行使其对相对人的权利
│     │     ③ 影响债权人债权的实现
│     │     ④ 该权利非专属于债务人自身
│     │
│     ├── 例外（536条）：债权未到期的特殊代位
│     │     └── 诉讼时效即将届满 / 未及时申报破产债权
│     │     └── 可代位请求相对人向债务人履行·申报债权等保全行为
│     │
│     ├── 行使效果（537条）
│     │     └── 代位权成立→相对人直接向债权人履行
│     │     └── 三方债权债务相应终止
│     │
│     └── 费用·范围
│           └── 以债权人到期债权为限；费用由债务人负担
│
├── 撤销权（538-542条）
│     │
│     ├── 针对：债务人【积极处分】——损害债权人利益的行为
│     │
│     ├── 无偿处分（538条）——无需相对人知情
│     │     ├── 放弃债权
│     │     ├── 放弃债权担保
│     │     ├── 无偿转让财产
│     │     └── 恶意延长到期债权履行期限
│     │
│     ├── 有偿但不合理处分（539条）——须相对人知道或应知
│     │     ├── 明显不合理低价转让财产
│     │     ├── 明显不合理高价受让财产
│     │     └── 为他人债务提供担保
│     │
│     ├── 行使范围·费用（540条）
│     │     └── 以债权人债权为限；费用由债务人负担
│     │
│     ├── 除斥期间（541条）
│     │     ├── 知道或应知撤销事由之日起1年内
│     │     └── 行为发生之日起5年内（绝对上限）
│     │
│     └── 效果（542条）：被撤销行为自始无法律约束力
│
└── 【两权对比】
      ├── 代位权：债务人消极→债权人积极替代行使
      ├── 撤销权：债务人积极（有害）→债权人请求法院否定
      └── 共同点：均以"影响债权实现"为触发条件；费用均由债务人负担



| 条文 | 核心功能 | 针对行为 | 保护对象 | 条件 / 门槛 |
| :--- | :--- | :--- | :--- | :--- |
| **535** | 代位权（到期） | 债务人怠于收债 | 债权人 | 四要件；非专属权利 |
| **536** | 代位权（保全） | 债务人怠于保全权利 | 债权人 | 时效将届满；未申报破产 |
| **537** | 代位权效果 | 三方清偿关系 | 债权人 | 直接履行给债权人 |
| **538** | 撤销权（无偿） | 无偿处分财产 | 债权人 | 无需相对人知情 |
| **539** | 撤销权（有偿） | 不合理有偿处分 | 债权人 | 相对人知道或应知 |
| **541** | 除斥期间 | 锁定行使期限 | 相对人·交易安全 | 1年（主观）+ 5年（客观） |
| **542** | 撤销效力 | 否定处分效果 | 债权人 | 溯及既往 |
