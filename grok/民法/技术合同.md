“先定范围与目的 → 再规范内容与支付 → 重点分职务非职务权属并保精神权利 → 最后设垄断无效红线”

技术合同（第一节总括）
│
├── 【类型】（843条）
│     技术开发合同·技术转让合同·技术许可合同·技术咨询合同·技术服务合同
│     └── 后续各节分别规定，本节是通则
│
├── 【技术成果归属】——核心权属体系
│     │
│     ├── 职务技术成果（847条）
│     │     ├── 定义：执行工作任务 或 主要利用单位物质技术条件
│     │     ├── 权属：使用权·转让权→单位
│     │     └── 完成人特权：同等条件下优先受让权
│     │
│     ├── 非职务技术成果（848条）
│     │     └── 权属：使用权·转让权→完成人个人
│     │
│     └── 精神权利（849条）——无论职务/非职务均保留
│           ├── 署名权（在成果文件上写明完成者）
│           └── 荣誉权（获得荣誉证书·奖励）
│
├── 【付款方式体系】（846条）
│     ├── 一次性付款
│     │     ├── 一次总算·一次总付
│     │     └── 一次总算·分期支付
│     └── 提成支付（核心创新机制）
│           ├── 计算基础：产品价格·新增产值·利润·销售额等
│           ├── 比例结构：固定比例·逐年递增·逐年递减
│           ├── 变体：提成+预付入门费
│           └── 配套权利：约定查阅会计账目
│
├── 【合同内容要件】（845条）
│     ├── 基本条款：项目名称·标的·履行计划·保密·成果归属·收益分配·验收标准
│     ├── 可纳入组成部分：技术文档·可行性报告·任务书等
│     └── 涉专利须注明：发明名称·申请人·专利权人·申请日期·申请号·专利号·有效期
│
└── 【效力底线】（850条）
      └── 无效情形：非法垄断技术 或 侵害他人技术成果

| **触发词**                  | **复述路径**                                                            |
| ------------------------ | ------------------------------------------------------------------- |
| **"在公司做的技术成果是自己的还是公司的"** | **847·848条二分**：执行工作任务或主要用单位资源 → **职务成果** → 单位所有；其他 → **非职务** → 个人所有 |
| **"公司卖掉职务成果，发明人有什么权利"**  | **两类**：财产权已归单位，但完成人有**同等条件优先受让权**（847）；**署名权和荣誉权**永远保留（849）         |
| **"技术合同怎么约定价格"**         | **一次性或提成**，提成可按产值/利润/销售额，比例可固定/递增/递减；约定提成可同时约定**查账权**（846）          |
| **"技术合同什么情况下无效"**        | **850条**：**非法垄断技术**或**侵害他人技术成果**；另适用民法典总则无效情形                       |
| **"涉及专利的技术合同要注意什么"**     | **845条**：须注明**发明名称·申请人·专利权人·申请日·申请号·专利号·有效期限**，缺一可能引发争议             |


法考/司法考试/法硕/考研等最常考的题型是：

“简述技术合同一般规定的主要内容 / 特征 / 基本原则”
“简述技术合同的立法目的与主要规则”
“技术合同一般规定体现了哪些特殊性”

考场思考路径（30秒内定框架）：
别死背条文序号，先问自己四个问题（按逻辑顺序）：

技术合同是什么？（范围+目的）
怎么订？（内容要求 + 支付方式）
技术成果归谁？（权属规则最重点，务必展开）
什么情况下无效？（底线）


按这个顺序分4—6点写，几乎能覆盖所有采分点。示例标准答案框架（控制在400—600字，写满6点最佳）：

定义与调整范围
技术合同是当事人就技术开发、转让、许可、咨询、服务五类所订立的合同（统一纳入，体现体系化）。
立法目的与基本原则
订立技术合同应当有利于知识产权保护、科学技术进步，促进成果研发、转化、应用、推广（体现强烈的公共政策导向，是技术合同区别于其他合同的最重要特征）。
合同内容的基本要求
合同一般应当包括项目名称、标的范围、保密、成果归属、收益分配、验收标准等条款；涉及专利的，必须详细注明专利基本信息（解决信息不对称问题）。
价款支付方式的特殊规定
允许一次总付、分期、提成支付等多种方式，特别鼓励提成支付并可约定查账（体现对技术价值不确定性的适应，激励技术供给方推动转化）。
技术成果权属与完成人权利（最重磅，务必展开2—3小点）
职务技术成果的使用、转让权归单位所有，但完成人享有优先受让权；
非职务技术成果的使用、转让权归完成人个人；
完成人享有署名权及获得荣誉、奖励的权利，此项精神权利不可剥夺（体现单位资源投入与个人创造贡献的利益平衡 + 对创新者的持续激励）。

合同无效的特别情形
非法垄断技术、侵害他人技术成果的合同或条款无效（维护技术市场公平竞争秩序，体现公序良俗限制）。

写满分的技巧：

每点开头用关键词（如“定义与范围”“立法目的”“成果权属规则”“支付方式的灵活性”“无效情形”）。
权属部分至少拆成职务/非职务/精神权利三小层，容易多得1—2分。
适当点出“与普通合同的区别”或“体现科技政策导向”，加印象分。
语言简洁，每点3—5句，逻辑词连接（首先、其次、再次、最后、特别体现）。
