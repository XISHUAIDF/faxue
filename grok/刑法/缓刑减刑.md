缓刑（母体）
　　├─ 适用条件（72条）：拘役/3年以下有期 + 四条件 / 特殊群体应当
　　├─ 附加禁止令（72条第2款）
　　├─ 考验期（73条）：拘役原刑-1年≥2月 / 有期原刑-5年≥1年
　　├─ 不适用（74条）：累犯 + 犯罪集团首要
　　├─ 义务（75条）：四项监督义务
　　├─ 社区矫正（76条） → 无撤销事由 → 原刑不再执行 + 公开宣告
　　└─ 撤销（77条）：新罪/漏罪 → 数罪并罚 / 违反监督/禁止令情节严重 → 执行原判
减刑（母体）
　　├─ 对象（78条）：管制/拘役/有期/无期
　　├─ 条件：悔改可以 / 立功可以 / 重大立功应当（六项）
　　├─ 最低执行期（78条第2款）：有期1/2、无期13年、死缓限制减刑25/20年
　　├─ 程序（79条）：执行机关建议 → 中院以上合议庭裁定 / 法定程序必经
　　└─ 起算（80条）：无期减有期从裁定日起

假释（母体）
　　├─ 适用条件（81条）：有期执行1/2以上、无期13年以上 + 悔改 + 无再犯危险 + 社区影响可控 / 特殊情况最高法核准例外
　　├─ 不适用（81条第2款）：累犯 + 八类暴力重罪判10年以上/无期
　　├─ 程序（82条）：参照减刑（执行机关建议 → 中院以上合议庭裁定）
　　├─ 考验期（83条）：有期剩余刑期 / 无期10年 / 从假释日起算
　　├─ 义务（84条）：四项监督义务（与缓刑相同）
　　├─ 社区矫正（85条） → 无撤销事由 → 原判刑罚视为执行完毕 + 公开宣告
　　└─ 撤销（86条）：新罪/漏罪 → 数罪并罚 / 违反监督规定 → 收监执行剩余刑期
