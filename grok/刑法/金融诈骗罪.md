金融诈骗罪（第五节总称）
│
├── 按【攻击对象/金融工具类型】分类
│     │
│     ├── 集资诈骗罪（192条）
│     │     └── 攻击对象：不特定公众的投资信心
│     │     └── 核心要件：非法占有目的 + 诈骗方法 + 非法集资
│     │
│     ├── 贷款诈骗罪（193条）
│     │     └── 攻击对象：银行/金融机构信贷体系
│     │     └── 五种手段列举（虚假理由/合同/证明/担保/其他）
│     │     └── 注意：只有自然人，单位不构成本罪（单位骗贷另行处理）
│     │
│     ├── 金融票据诈骗罪（194条）
│     │     └── 攻击对象：票据流通信用（汇票/本票/支票）
│     │     └── 扩展：银行结算凭证（委托收款/汇款凭证/存单）
│     │
│     ├── 信用证诈骗罪（195条）
│     │     └── 攻击对象：国际贸易信用证体系
│     │
│     ├── 信用卡诈骗罪（196条）
│     │     ├── 四种手段：伪造卡/骗领卡/作废卡/冒用/恶意透支
│     │     ├── 恶意透支：特别定义条款（目的+超限+催收后不还）
│     │     └── 特别规定：盗窃+使用→按盗窃罪定罪（不按本条）
│     │
│     ├── 有价证券诈骗罪（197条）
│     │     └── 攻击对象：国库券/国家有价证券
│     │
│     └── 保险诈骗罪（198条）
│           ├── 主体：投保人/被保险人/受益人
│           ├── 五种手段（虚构标的→夸大损失→虚构事故→故意毁财→故意伤害/杀人）
│           ├── 第4、5项：数罪并罚
│           ├── 单位犯罪：另有量刑标准
│           └── 共犯扩展：鉴定人/证明人/评估人提供虚假材料→共犯
│
├── 【刑罚梯度】——各罪通用结构（除保险诈骗罚金略低）
│     ├── 数额较大 → 5年以下/拘役 + 罚金
│     ├── 数额巨大/严重情节 → 5-10年 + 罚金
│     └── 数额特别巨大/特别严重 → 10年以上/无期 + 罚金或没收财产
│           └── 集资诈骗（192）起点更高：较大→3-7年
│
└── 【单位犯罪】
      ├── 192（集资诈骗）：单位可构成
      ├── 193（贷款诈骗）：单位不构成（立法空白，实践争议）
      ├── 194/195：单位可构成（200条统一规定）
      └── 198（保险诈骗）：单位可构成（198条内单独规定）

## grok：
金融诈骗罪（第五节总称，第192-200条）
│
├── 【按攻击对象/金融工具类型分类】
│   ├── 集资诈骗罪（192条）
│   │   ├── 攻击对象：不特定公众的投资信心 + 集资秩序
│   │   ├── 核心要件：非法占有目的 + 使用诈骗方法 + 非法集资
│   │   ├── 量刑：数额较大 → 3年以上7年以下 + 罚金；巨大/严重情节 → 7年以上/无期 + 罚金/没收
│   │   └── 单位犯罪：可构成，双罚制（单位罚金 + 责任人员同前款）
│   │
│   ├── 贷款诈骗罪（193条）
│   │   ├── 攻击对象：银行/金融机构信贷体系
│   │   ├── 核心要件：非法占有目的 + 诈骗贷款 + 五种手段列举（虚假理由/合同/证明/担保/其他）
│   │   ├── 量刑：较大 → 5年以下/拘役 + 2-20万罚金；巨大/严重 → 5-10年 + 5-50万罚金；特别巨大/特别严重 → 10年以上/无期 + 5-50万罚金或没收
│   │   └── 主体特殊：仅自然人（单位不构成本罪，实践常转合同诈骗罪）
│   │
│   ├── 金融票据诈骗罪（194条）
│   │   ├── 攻击对象：汇票/本票/支票 + 银行结算凭证信用（委托收款/汇款/存单）
│   │   ├── 五种手段：伪造变造使用/作废使用/冒用/空头支票/无资金虚假记载
│   │   └── 量刑：较大 → 5年以下/拘役 + 2-20万罚金；巨大/严重 → 5-10年 + 5-50万罚金；特别巨大/特别严重 → 10年以上/无期 + 5-50万罚金或没收
│   │
│   ├── 信用证诈骗罪（195条）
│   │   ├── 攻击对象：国际贸易信用证体系
│   │   ├── 四种手段：伪造变造使用/作废使用/骗取信用证/其他
│   │   └── 量刑：同194条（较大5年以下...）
│   │
│   ├── 信用卡诈骗罪（196条）
│   │   ├── 攻击对象：信用卡支付信用
│   │   ├── 四种手段：伪造卡使用 / 骗领卡使用 / 作废卡使用 / 冒用他人卡 / 恶意透支（定义：非法占有目的 + 超限/超期 + 催收后不还）
│   │   ├── 特别规定：盗窃信用卡并使用 → 按盗窃罪（264条）定罪（不按本罪）
│   │   └── 量刑：同194条（较大5年以下...）
│   │
│   ├── 有价证券诈骗罪（197条）
│   │   ├── 攻击对象：国库券/国家其他有价证券
│   │   └── 量刑：同194条（较大5年以下...）
│   │
│   └── 保险诈骗罪（198条）
│       ├── 主体：投保人/被保险人/受益人（特殊）
│       ├── 五种手段：虚构标的 / 编造原因/夸大损失 / 编造事故 / 故意毁财 / 故意伤害/杀人
│       ├── 特别：第4、5项同时构成其他犯罪 → 数罪并罚
│       ├── 共犯：鉴定人/证明人/评估人提供虚假证明 → 以本罪共犯论
│       ├── 量刑：较大 → 5年以下/拘役 + 1-10万罚金；巨大/严重 → 5-10年 + 2-20万罚金；特别巨大/特别严重 → 10年以上 + 2-20万罚金或没收（罚金起点/幅度最低）
│       └── 单位犯罪：单独规定（较自然人轻：5年以下/拘役；巨大5-10年；特别巨大10年以上）
│
├── 【刑罚梯度】——通用三档（除集资起点更高、保险罚金更低）
│   ├── 数额较大 → 多数5年以下/拘役 + 罚金（集资3-7年）
│   ├── 数额巨大/严重情节 → 5-10年 + 罚金
│   └── 数额特别巨大/特别严重情节 → 10年以上/无期 + 罚金或没收
│
└── 【单位犯罪特殊规定】
    ├── 192条（集资）：可单位，双罚（同自然人）
    ├── 193条（贷款）：不可单位（无规定，实践转合同诈骗）
    ├── 194、195条：可单位（200条统一：5年以下/拘役，可并罚金；巨大5-10年+罚金；特别巨大10年以上/无期+罚金）
    └── 198条（保险）：可单位，单独量刑（较轻）
    （注：199条已删去，无死刑条款）
