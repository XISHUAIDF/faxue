│
├── 按【知识产权类型】分
│     │
│     ├── 商标权犯罪（三条，对应产业链三环节）
│     │     ├── 213条：使用假冒商标（生产/销售端·使用行为）
│     │     │     └── 要件：同一商品+相同商标+未经许可+情节严重
│     │     ├── 214条：销售假冒商标商品（流通端·销售行为）
│     │     │     └── 要件：明知+销售+违法所得数额
│     │     └── 215条：伪造/制造商标标识（制造端·标识本身）
│     │           └── 要件：伪造/擅自制造或销售该标识+情节严重
│     │
│     ├── 专利权犯罪
│     │     └── 216条：假冒他人专利
│     │           └── 注意：只保护"假冒"，不保护专利技术本身被复制
│     │           └── 最高刑：3年/拘役（全节最低，无第二档）
│     │
│     ├── 著作权犯罪（两条）
│     │     ├── 217条：侵犯著作权罪（制作/传播端）
│     │     │     ├── 前提：以营利为目的
│     │     │     ├── 六种行为（复制发行·网络传播·专有出版·录音录像·表演·技术措施）
│     │     │     └── 第六项新增：破坏技术保护措施（反规避条款）
│     │     └── 218条：销售侵权复制品罪（流通端）
│     │           └── 注意：起点是"数额巨大"，门槛高于217条
│     │           └── 最高刑：5年（低于217条的10年）
│     │
│     └── 商业秘密犯罪（两条）
│           ├── 219条：侵犯商业秘密罪（一般情形）
│           │     ├── 三种获取手段（盗窃/贿赂/欺诈/胁迫/电子侵入）
│           │     ├── 知情使用者同等处理（明知故用）
│           │     └── 权利人扩展：所有人+被许可使用人
│           └── 219条之一：为境外提供商业秘密（涉外加重）
│                 └── 起点刑：5年以下（不同于219条的情节严重标准）
│                 └── 背景：国家安全视角，不仅是商业利益保护
│
├── 【刑罚结构对比】
│     ├── 商标三罪·219·217：情节严重→3年以下；特别严重→3-10年
│     ├── 216（假冒专利）：最高3年/拘役，无第二档（保护力度最弱）
│     ├── 218（销售侵权复制品）：最高5年（中等）
│     └── 219之一（境外商业秘密）：最高无期（涉国安，另行处理）
│
└── 【单位犯罪】（220条统一规定）
      └── 213-219之一均可构成单位犯罪
      └── 单位：罚金；责任人：按各条自然人刑期

侵犯知识产权罪（第七节，第213-220条）
│
├── 【按知识产权类型 + 产业链环节分类】
│   ├── 商标权犯罪（213-215条，全链条打击）
│   │   ├── 213条：假冒注册商标罪（生产/销售端·使用行为）
│   │   │   ├── 要件：未经许可 + 同一种商品/服务 + 使用相同商标 + 情节严重
│   │   │   └── 量刑：情节严重 → 3年以下 + 罚金/单罚；特别严重 → 3-10年 + 罚金
│   │   ├── 214条：销售假冒注册商标的商品罪（流通端·销售行为）
│   │   │   ├── 要件：明知 + 销售假冒注册商标商品 + 违法所得较大/其他严重情节
│   │   │   └── 量刑：违法所得较大/严重情节 → 3年以下 + 罚金/单罚；巨大/特别严重 → 3-10年 + 罚金
│   │   └── 215条：非法制造/销售注册商标标识罪（制造端·标识本身）
│   │       ├── 要件：伪造/擅自制造或销售他人注册商标标识 + 情节严重
│   │       └── 量刑：同213条（严重3年以下，特别严重3-10年 + 罚金）
│   │
│   ├── 专利权犯罪
│   │   └── 216条：假冒专利罪
│   │       ├── 要件：假冒他人专利（冒充专利号/标记） + 情节严重
│   │       ├── 注意：不保护专利技术本身被复制/实施（行政执法为主）
│   │       └── 量刑：3年以下或拘役 + 罚金/单罚（全节最高刑最低，无第二档）
│   │
│   ├── 著作权犯罪（217-218条）
│   │   ├── 217条：侵犯著作权罪（制作/传播端）
│   │   │   ├── 前提：以营利为目的
│   │   │   ├── 六种行为（完整列举）：
│   │   │   │   （一）未经许可复制发行/信息网络传播文字、音乐、美术、视听、计算机软件及其他作品
│   │   │   │   （二）出版他人专有出版权的图书
│   │   │   │   （三）未经许可复制发行/网络传播录音录像制品
│   │   │   │   （四）未经许可复制发行/网络传播表演的录音录像，或传播表演
│   │   │   │   （五）制作/出售假冒他人署名的美术作品
│   │   │   │   （六）故意避开/破坏技术保护措施（反规避条款，含间接提供规避工具/服务）
│   │   │   └── 量刑：违法所得较大/严重情节 → 3年以下 + 罚金/单罚；巨大/特别严重 → 3-10年 + 罚金
│   │   └── 218条：销售侵权复制品罪（流通端）
│   │       ├── 要件：以营利为目的 + 明知 + 销售217条侵权复制品 + 违法所得巨大/其他严重情节
│   │       ├── 注意：入罪门槛高于217条（数额巨大起）
│   │       └── 量刑：5年以下 + 罚金/单罚（最高5年，中等保护力度）
│   │
│   └── 商业秘密犯罪（219-219之一条）
│       ├── 219条：侵犯商业秘密罪（一般情形）
│       │   ├── 三种行为：
│       │   │   （一）不正当手段获取（盗窃/贿赂/欺诈/胁迫/电子侵入/其他）
│       │   │   （二）披露/使用/允许他人使用前项获取的商业秘密
│       │   │   （三）违反保密义务/要求，披露/使用/允许他人使用掌握的商业秘密
│       │   ├── 扩展：明知前款行为而获取/披露/使用/允许他人使用 → 同罪
│       │   ├── 权利人：所有人 + 经许可的使用人
│       │   └── 量刑：情节严重 → 3年以下 + 罚金/单罚；特别严重 → 3-10年 + 罚金
│       └── 219条之一：为境外窃取/刺探/收买/非法提供商业秘密罪（涉外加重）
│           ├── 要件：为境外的机构/组织/人员实施上述行为
│           ├── 起点：5年以下 + 罚金/单罚（无“情节严重”前置，门槛低）
│           ├── 情节严重 → 5年以上 + 罚金
│           └── 背景：国家安全优先（可达无期结合其他罪）
│
├── 【刑罚梯度对比】
│   ├── 多数罪（213/214/215/217/219）：情节严重 → 3年以下；特别严重 → 3-10年 + 罚金
│   ├── 216条（假冒专利）：最高3年/拘役，无第二档（保护最弱）
│   ├── 218条（销售侵权复制品）：最高5年（中等）
│   └── 219条之一（境外商业秘密）：5年以下起跳，情节严重5年以上（最高可无期，保护最强）
│
└── 【单位犯罪】（220条统一规定）
    └── 213-219之一全覆盖单位犯罪
    └── 双罚制：单位罚金 + 责任人员按各条自然人量刑
