法人（母体）
　　├─ 营利法人（以利润分配为目的）
　　├─ 非营利法人（母体）
	　　├─ 定义（87条）：公益/非营利目的 + 不分配利润
	　　├─ 事业单位法人（88-89条）
	　　│    └─ 公益服务 + 登记/自动资格 → 理事会决策 + 法定代表人灵活产生
	　　├─ 社会团体法人（90-91条）
	　　│    └─ 会员共同意愿 + 公益/互益 → 会员大会权力 + 理事会执行 + 理事长/会长法定代表人
	　　└─ 捐助法人（92-95条）
	　　　　 └─ 捐助财产 + 公益目的 → 基金会/社会服务机构/宗教场所可申请 → 理事会/民主管理决策 + 执行 + 监事会监督 → 捐助人查询/撤销权 → 剩余财产公益化（禁止私分 + 转同类 + 公告）
	　　　　
　　└─ 特别法人（中国特色，功能特殊，无法归入前二类）
　　　　├─ 机关法人（97-98条）
　　　　│    └─ 独立经费 + 行政职能法定机构 → 自动资格 → 限于职能需要民事活动 → 撤销承继（继任优先 → 决定机关）
　　　　├─ 农村集体经济组织法人（99条）
　　　　│    └─ 依法取得 → 优先特别法/行政法规 → 管理集体财产/发展经济
　　　　├─ 城镇农村合作经济组织法人（100条）
　　　　│    └─ 依法取得 → 优先特别法/行政法规 → 成员互助/对外经营
　　　　└─ 基层群众性自治组织法人（101条）
　　　　　　 └─ 居/村委会 → 限于职能需要民事活动 → 未设村集体经济组织 → 村委会代行其职能
