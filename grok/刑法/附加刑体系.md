附加刑体系
│
├── 剥夺政治权利（第七节）
│     │
│     ├── 内容（54条）——四类权利
│     │     ├── 选举权·被选举权（政治参与核心）
│     │     ├── 言论·出版·集会·结社·游行·示威自由
│     │     ├── 担任国家机关职务权
│     │     └── 担任国有单位·人民团体领导职务权
│     │
│     ├── 期限体系
│     │     ├── 一般：1-5年（55条）
│     │     ├── 管制附加：与管制期限相等·同时执行（55条）
│     │     ├── 死刑·无期：终身（57条）
│     │     └── 死缓/无期减为有期后：改为3-10年（57条）
│     │
│     ├── 适用方式
│     │     ├── 附加适用：随主刑一并判处
│     │     │     ├── 应当附加：危害国家安全犯罪（56条）
│     │     │     └── 可以附加：故意杀人·强奸·放火·爆炸·投毒·抢劫等（56条）
│     │     └── 独立适用：依分则规定
│     │
│     └── 执行规则（58条）
│           ├── 起算：主刑执行完毕或假释之日
│           ├── 主刑期间：自然适用，不另计
│           └── 执行期间义务：遵守监管规定·服从监督·不得行使54条各项权利
│
└── 没收财产（第八节）
      │
      ├── 范围（59条）
      │     ├── 一部或全部没收
      │     ├── 全部没收时：保留本人及扶养家属必需生活费用
      │     └── 禁止没收：家属所有或应有的财产
      │
      └── 债务处理（60条）
            └── 没收前的正当债务→经债权人请求→应当偿还
            └── 逻辑：合法债权优先于国家没收权


| 条文     | 社会功能      | 解决的问题/保护谁 | 核心机制           |
| ------ | --------- | --------- | -------------- |
| **54** | 定义政治权利范围  | 政治秩序      | 四类权利清单化        |
| **55** | 期限上下限     | 被告人       | 防止无限期剥夺        |
| **56** | 适用场景区分    | 社会秩序      | 国安罪应当；暴力犯罪可以   |
| **57** | 死刑/无期对应终身 | 社会        | 重罪与政治权利终身挂钩    |
| **58** | 起算时间规则    | 逻辑自洽      | 主刑期间自然适用，出狱后另计 |
| **59** | 没收范围+家属保护 | 无辜家属      | 保留生活费；不及家属财产   |
| **60** | 债务优先偿还    | 债权人       | 合法债权不因没收而消灭    |
