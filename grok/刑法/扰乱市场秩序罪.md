刑法第八节：扰乱市场秩序罪（§221-§231）
├── 立法背景与总体框架（社会功能：保护市场信息对称、公平竞争、自愿交易）
│   ├── 立法目的：刑事兜底民事救济不足，打击欺骗、串通、垄断、暴力等破坏市场规则行为
│   └── 五类攻击面分类（按破坏方式）
│       ├── 信息欺骗类（§221、§222、§229、§230）：虚假信息/证明破坏信任
│       ├── 合同诈骗类（§224、§224之一）：利用合同骗取财物/传销
│       ├── 竞争扭曲类（§223、§226）：串通/强迫扭曲竞争
│       ├── 无证经营类（§225、§228）：违反准入许可扰乱管控
│       └── 票证造假类（§227）：伪造/倒卖票证破坏交易工具
├── 各罪详解（按条文顺序：立法目的 + 构成要件 + 裁定性要点）
│   ├── §221 损害商业信誉、商品声誉罪（信息武器化滥用）
│   │   ├── 构成要件
│   │   │   ├── 行为：捏造 + 散布虚伪事实
│   │   │   ├── 对象：他人商业信誉/商品声誉
│   │   │   ├── 结果：重大损失/其他严重情节
│   │   │   └── 主观：故意
│   │   └── 裁定性要点：需同时"捏造+散布"；散布真实负面不构罪（可能民事侵权）
│   ├── §222 虚假广告罪（制造信息不对称侵害消费者）
│   │   ├── 构成要件
│   │   │   ├── 主体：广告主/经营者/发布者（三类，可单独入罪）
│   │   │   ├── 行为：违反规定虚假宣传商品/服务
│   │   │   └── 结果：情节严重
│   │   └── 裁定性要点：主体特定，共同犯罪常见
│   ├── §223 串通投标罪（架空竞争机制）
│   │   ├── 两种串通类型
│   │   │   ├── 类型1：投标人之间串通（损害招标人/其他投标人）
│   │   │   └── 类型2：投标人与招标人串通（损害国家/集体/公民利益）
│   │   └── 裁定性要点：两款罪名/量刑相同，但损害对象不同
│   ├── §224 合同诈骗罪（破坏合同信任基础）
│   │   ├── 五种行为方式
│   │   │   ├── 1. 虚构主体/冒用名义签合同
│   │   │   ├── 2. 虚假担保（伪造票据/产权证明）
│   │   │   ├── 3. 无履行能力诱骗续约（先小额履行）
│   │   │   ├── 4. 收款/预付款后逃匿
│   │   │   └── 5. 其他欺骗（兜底）
│   │   ├── 量刑三档（基于数额/情节）
│   │   │   ├── 较大：3年以下 + 罚金
│   │   │   ├── 巨大/严重：3-10年 + 罚金
│   │   │   └── 特别巨大/特别严重：10年以上/无期 + 罚金/没收财产
│   │   └── 裁定性要点：主观非法占有目的核心；第三方式易混淆（无履行能力）
│   ├── §224之一 组织、领导传销活动罪（庞氏骗局扰乱社会）
│   │   ├── 四要素
│   │   │   ├── 1. 以推销为名（幌子）
│   │   │   ├── 2. 缴费/购买获资格（准入壁垒）
│   │   │   ├── 3. 层级发展计酬（金字塔）
│   │   │   └── 4. 引诱/胁迫发展他人（传导）
│   │   └── 裁定性要点：主体仅组织/领导者；参与者不构罪（行政违法）
│   ├── §225 非法经营罪（破坏市场准入管控）
│   │   ├── 四种行为
│   │   │   ├── 1. 无许可经营专营/专卖/限制物品
│   │   │   ├── 2. 买卖经营许可文件
│   │   │   ├── 3. 无批准经营证券/期货/保险/资金结算
│   │   │   └── 4. 其他严重扰乱（兜底）
│   │   └── 裁定性要点：第三项金融类常见；罚金1-5倍违法所得
│   ├── §226 强迫交易罪（破坏交易自愿性）
│   │   ├── 五种行为（均需暴力/威胁手段）
│   │   │   ├── 强买强卖
│   │   │   ├── 强迫提供/接受服务
│   │   │   ├── 强迫参与/退出招投标
│   │   │   ├── 强迫转让/收购股权/债券/资产
│   │   │   └── 强迫参与/退出特定经营
│   │   └── 裁定性要点：暴力/威胁是关键（区别民事纠纷）
│   ├── §227 票证罪（两款独立）
│   │   ├── 第一款：伪造/倒卖伪造有价票证（涉假票）
│   │   └── 第二款：倒卖真实车票/船票（黄牛，量刑轻）
│   ├── §228 非法转让、倒卖土地使用权罪（违反土地管控）
│   │   ├── 构成要件
│   │   │   ├── 主观：牟利目的
│   │   │   ├── 行为：违反法规非法转让/倒卖
│   │   │   └── 结果：情节严重
│   │   └── 裁定性要点：罚金5%-20%价额
│   ├── §229 中介虚假证明罪（摧毁市场信任基础设施）
│   │   ├── 两罪名
│   │   │   ├── 提供虚假证明文件罪（故意）：5年以下；加重5-10年（三种情形）
│   │   │   │   └── 三加重：证券发行/重大资产交易特别严重；公共安全工程致特别重大损失
│   │   │   └── 出具证明文件重大失实罪（过失）：3年以下
│   │   └── 裁定性要点：收受财物竞合择一重罪
│   ├── §230 逃避商品检验罪（破坏边境控制）
│   │   └── 两种行为：进口未经检验销售/使用；出口未经合格出口
│   └── §231 单位犯罪（双罚制兜底）
│       ├── 单位：罚金
│       └── 负责人/责任人：依各条处罚
├── 考场可背诵版（高频考点对比）
│   ├── 总体框架：五类行为 + 单位兜底
│   ├── 各罪核心要件（主体-主观-行为-结果-量刑）
│   ├── 高频对比表（诋毁/合同骗/传销/强迫/中介造假/票证）
│   └── 简述题框架（主体→主观→行为→结果→量刑→特殊说明）
├── 默写句式提示（复述训练）
│   ├── 模板：主体 + 主观目的 + 客观行为 + 结果 + 量刑
│   └── 各罪主线示例（221捏造散布；224五种欺骗非法占有；等）
└── 考场简述题思考顺序（总体）
    ├── 1. 定性：保护法益（市场秩序/交易安全/信息真实）
    ├── 2. 分类：按行为归类
    ├── 3. 逐类展开：四要素（主体-主观-行为-结果）
    ├── 4. 特殊要件：非法占有/暴力/故意vs过失
    └── 5. 单位兜底：双罚制
