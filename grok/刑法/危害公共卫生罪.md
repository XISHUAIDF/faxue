│     │     ├── 331条：传染病菌种毒种扩散罪
│     │     │     ├── 主体：特定人员（实验·保藏·携带·运输者）
│     │     │     └── 门槛：造成扩散+后果严重（实害犯）

│     │     ├── 333条：非法组织卖血罪
│     │     │     ├── 组织他人卖血：5年以下
│     │     │     ├── 强迫他人卖血：5-10年
│     │     │     └── 造成伤害：按故意伤害罪（234条）并罚
│     │     └── 334条：采供血及血液制品事故罪
│     │           ├── 非法采供血/制作血液制品不达标：三级刑罚
│     │           └── 合法机构违规操作造成危害：单位罚金+责任人5年以下

│     │     └── 334条之一：非法采集·出境人类遗传资源罪
│     │           ├── 背景：国家主权+生物安全
│     │           └── 两种行为：非法采集 / 非法出境

│     │           ├── 主体：未取得医生执业资格的人

| 條文 | 罪名/對象 | 入罪門檻 | 特殊設計（法律邏輯） | 社會功能 / 第一性原理 |
| --- | --- | --- | --- | --- |
| **330** | **傳染病防治** | **危險犯**：有傳播嚴重危險即入罪 | 列舉五種行為（如拒絕隔離、違規排污） | **預判轉角後的危機**：在大流行病爆發前切斷傳播鏈。 |
| **331** | **菌種毒種管理** | **實害犯**：造成擴散 + 後果嚴重 | 針對**特定職業主體**（實驗室、科研人員） | 專業監管：防止高危生物樣本外流。 |
| **332** | **國境衛生檢疫** | **危險犯**：引起檢疫傳染病傳播危險 | 聚焦**跨境傳輸**場景（海關、邊境） | 國境防火牆：維護國家生物主權與安全。 |
| **333** | **非法組織/強迫賣血** | **行為犯**：有組織行為即入罪 | 造成傷害則轉致（加重）為**故意傷害罪** | 倫理底線：禁止將人體血液商品化。 |
| **334** | **採供血安全** | **雙軌制**：危險犯 + 實害犯 | 區分合法機構與非法機構，非法者從重 | 生命線防護：確保血液製品的純淨。 |
| **334-1** | **人類遺傳資源** | **情節嚴重** | 涉及國家主權 + 生物安全雙重定性 | 數據主權：防止基因樣本流向境外。 |
| **335** | **醫療事故** | **實害犯**：嚴重不負責任 + 損害後果 | 最高刑期僅 3 年，**保留醫療容錯空間** | 平衡保護：既懲戒失職，也避免醫師「防禦性醫療」。 |
| **336** | **非法行醫** | **三級制**：情節嚴重 $\rightarrow$ 實害 $\rightarrow$ 死亡 | **無執照即非法**，對地下診所零容忍 | 准入機制：將「無照運作」本身視為最大風險。 |
| **336-1** | **非法基因編輯/克隆** | **情節嚴重** | 守護**生命倫理底線**，將科學倫理法律化 | 邊界守護：防止對人類遺傳特徵進行不可逆修改。 |
| **337** | **動植物防疫** | **危險犯 + 情節嚴重** | 引起重大動植物疫情（如非洲豬瘟） | 間接保護：透過保護產業鏈來守護人類食安。 |
