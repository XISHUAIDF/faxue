仅考1237 编（总则、物权、合同、侵权责任），其中物权编（担保制度）+ 合同编（担保合同） 是绝对核心，总则和侵权责任考查极少。
3. 关键法条提醒
    民法典 386 条（担保物权开篇）：需逐字研读，明确除留置权（法定担保物权）外，所有担保均需书面担保合同（抵押权、质权、保证、非典型担保均需）；
    民法典 388 条：担保合同的核心依据，是区分担保成立的关键；
    民法典 392 条：混合担保（物保 + 人保）核心规则；
    民法典 1064 条：夫妻共同债务认定（主观题重点考共同经营的界定）；
    民法典 552 条：债务加入（与连带责任保证极易混淆，是高频考点）。


主体切换：从客观题的自然人，转为主观题的商事主体（公司为主），核心考查公司担保问题（公司对外担保的决议前置、规避情形是高频考点）；
主体细分考查：公司担保会结合担保公司 / 上市公司 / 非上市公司、总公司 / 分公司、母公司 / 子公司的不同规则考查，需区分不同主体的担保要求；


四、五大核心考点（主观题民法高频必考，占比极高）
这五个考点是 2020 年后法考主观题的稳定考查重点，需围绕其深挖细节、结合法条和司法解释吃透，具体如下：
    民事责任：几乎每年必考，核心是违约责任（有合同且生效），其次是**缔约过失责任（有合同但未成立 / 无效）**、侵权责任（无合同）；判断关键：是否有合同关系。
    保证：每年必考，重点是连带责任保证（客观题考一般保证，主观题几乎不考）；核心难点：区分**连带责任保证与债务加入**（并存的债务承担），且债务加入类推适用连带责任保证规则。
    债的**保全**：天然结合民诉考查，核心是债权人代位权诉讼、债权人撤销权诉讼（均只能通过诉讼行使）；是实体 + 程序结合的典型考点，需熟悉诉讼相关规则。
    法人人格否认：考查核心是**法定连带责任**（与保证 / 债务加入的约定连带责任对应）；主观题常考 “甲、乙是否需对丙承担连带责任”，法人人格否认是法定连带责任的高频情形。
    **让与担保**（非典型担保）：名为让与、实为担保，是主观题难点；考查标的物包括不动产、动产、股权、应收账款，且《民法典合同编通则司法解释》将履行期限届满前的以物抵债协议认定为让与担保，是 2025 年新增重点。


    每月模考适应长案例（1500 字以上）、多当事人、多管辖地的主观题题型，克服 “长题综合症”。
    法条学习方法：每天翻看法条（无需背诵），参与 “数金条” 法条带读，重点熟悉民法典担保物权编、合同通则编及四大司法解释的条文位置和核心内容。
    2025 年重点关注侵权责任编的用工责任、夫妻共同债务的 “共同经营” 界定、股权的夫妻共同共有与无权处分 / 善意取得、履行期限届满前的以物抵债（让与担保）。


主观题作答需以法条为依据，无需死记硬背，但要熟悉条文位置和核心内容；民法典核心编 + 四大司法解释是重中之重，远重要于民法典本身的基础条文。
输出比输入更重要，**完成比完美更重要**：听课仅需 1 遍（可倍速），核心在于狂做题、多输出

### 民事法律关系三核心基本点（主体 / 客体 / 内容）法考考点精编
1.民事法律关系的主体，指参加民事法律关系，享有民事权利并承担民事义务的人（从权利义务角度定义，与客体定义逻辑一致）。
2.民法主体框架：自然人 + 组织（法人 + 非法人组织），国家 / 集体仅为特殊权利（如所有权）主体，简答需**拓展法人 / 非法人组织概念**（**核心概念（三关键词：主体资格 + 组织 + 核心区别）** 自然人无需定义，仅需说明 “从出生到死亡具有民事权利能力，具备主体资格”）。
- 法人	具有民事权利能力和民事行为能力的**组织**，能独立享有民事权利、承担民事义务	独立主体 + 独立责任，组织体人格独立
- 非法人组织	可以自己的名义从事民事活动的组织，不具有**法人资格**，不能独立承担**民事责任**	名义独立 + 责任不独立，财产不足以清偿债务时，出资人 / 设立人承担无限责任
3.民事法律客体
民事法律关系的客体，指民事权利和民事义务**所指向的对象**，本质是民事利益，其类型以民事权利的类型为展开线索（人身权→财产权→其他权利）。
-客体的具体类型（简答核心，按权利线索记）
 按 “人身利益 - 财产利益 - 综合性利益” 分类，所有术语均为**专业采分点**，需精准记忆，分则会反复考查：
 - 人身权（人格权 + 身份权）	人身利益	人格权→人格利益；身份权→身份利益
 - 财产权（物权 + 债权）	财产利益	物权→特定的物 + 法律规定的权利；债权→债务人的特定给付行为（核心学术用语）
 - 其他权利（知识产权 + 继承权）	综合性利益	知识产权→智力成果；继承权→遗产


#### 结果加重犯的核心特征
结果加重犯，是指行为人实施了基本犯罪行为，却过失地引发了更严重的后果，刑法因此规定了更重法定刑的情形 。其核心特征包括：
- 法定性：必须由刑法分则明文规定。这是结果加重犯存在的法律基础。
- 单一行为：只实施了一个基本犯罪行为（如故意伤害）
- 加重结果：基本行为过失导致了更严重的结果（如死亡）。
- 因果关系：基本行为与加重结果之间必须存在直接因果关系
