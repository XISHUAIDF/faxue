# 📊 比特幣 Cointime Price 定價模型筆記

## 1. 核心理念：時間加權
不同於傳統模型（如市值或單純的實現價格），**Cointime Price** 是針對比特幣獨有的 **UTXO 結構** 設計，核心在於**「時間加權」**。它認為持有的時間愈長，該單位 BTC 所承載的「經濟權重」就愈高。

---

## 2. 三大基礎指標 (CBC, CBD, CBS)

模型透過以下三個概念來衡量網絡的「經濟壽命」：

| 指標      | 全稱                        | 定義 / 邏輯                                |
| :------ | :------------------------ | :------------------------------------- |
| **CBC** | **Coin Blocks Created**   | 第 N 個區塊時的流通 BTC 總量。                    |
| **CBD** | **Coin Blocks Destroyed** | 會將轉移的 BTC 數量乘上這些 BTC 持有的時間（轉移前經過幾個區塊）， |
| **CBS** | **Coin Blocks Stored**    | **CBC - CBD**。目前處於靜止狀態的 BTC 時間權重。      |

---

## 3. Cointime Price 計算原理

### 公式邏輯
$$Cointime\ Price = \frac{Cointime\ Value\ Destroyed}{CBS}$$

* **分子 (Cointime Value Destroyed)**：而圖一公式中分子的 Cointime Value Destroyed，
* 則是再將每次轉移時產生的 CBD，乘上轉移時的價格，可得出「轉移（交易）時的 U 本位價值」 
* **分母 (CBS)**：現存未花費的時間權重。

### 白話解讀
這可以視為**「場上籌碼在考慮時間加權後的平均成本」**。

---

## 4. 模型三大特性

1.  **反映長線動向**：因為時間加權的設計，當長期持有者（LTH）大量轉移（派發）時，CBD 劇增，Cointime Price 變化速度會明顯上升。
2.  **真實籌碼成本**：公式分子可視為目前場上「時間加權後的花費總額」，除以 CBS 後，即可得出市場參與者的平均成本。
3.  **排除無效數據**：因為 CBD 考慮了轉移行為，不轉移就不會產生 CBD，因此自動過濾了早期丟失、永久沉睡的「死幣」影響。

---

> **💡 總結**：Cointime Price 比 Realized Price 更能體現「老手」的動作。當綠線（Cointime Price）陡峭上揚時，通常代表長線資金正在大量換手或離場。
